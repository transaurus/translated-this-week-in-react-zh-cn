- 新架构默认启用（新项目），现有项目可手动升级。Expo Go现仅支持新架构

- 通过`'use dom'`指令实现DOM组件，支持渐进式迁移或无缝集成WebView
- React Native DevTools支持
- 通用Tree Shaking（实验性）
- React编译器（实验性）
