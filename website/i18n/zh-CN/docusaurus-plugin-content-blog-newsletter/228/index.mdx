## ⚛️ React {#react}

[![React 19.1](/emails/issues/228/react_19_1.jpg)](https://github.com/facebook/react/releases/tag/v19.1.0)

**[React 19.1](https://github.com/facebook/react/releases/tag/v19.1.0)**

React 19.1 本周发布，最显著的变化包括：

- 新增 [`captureOwnerStack()`](https://react.dev/reference/react/captureOwnerStack) API，框架将利用它在开发环境中提供更好的错误信息。它在处理 `children` 和 DOM 节点时与现有的组件堆栈不同。
- Parcel 2.14 中新增的服务器组件支持，与 React 的这部分功能协同工作。
- 实验性的 `unstable_prerender()`，用于在服务器上预渲染 React 服务器组件。

此版本还包括一系列次要更改和修复，其中大部分与服务器组件或 `Suspense` 边界相关。
