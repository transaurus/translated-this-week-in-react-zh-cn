## ⚛️ React {#react}

[![Next.js PR - 支持 Node.js 中间件](/emails/issues/220/nextjs-middleware.jpg)](https://github.com/vercel/next.js/pull/75624)

**[Next.js PR \- 支持 Node.js 中间件](https://github.com/vercel/next.js/pull/75624)**

Next.js 社区期待已久的功能。最新的 v15.2 canary 版本新增了 `experimental.nodeMiddleware` 标志，让你可以选择 `runtime: ‘nodejs’`，从而在中间件中使用 Node.js API，并摆脱仅限于与边缘运行时兼容的 API 的限制。

Vercel 中间件也已 [从头重建](https://x.com/galstar/status/1886832573482934706)，以支持多运行时并在 [Fluid compute](https://vercel.com/blog/introducing-fluid-compute) 上运行，这是 Vercel 运行无服务器函数的一种新的成本效益高的方式。

一个强大的React Native库，用于通过拖放手势重新排序项目，功能包括：

- 支持网格和弹性布局
- 支持不同项目大小、间距和排列方式
- 拖拽超出屏幕时自动滚动
- 流畅的布局动画
- 简洁的API，完整的TypeScript支持
- 兼容Expo和两种架构
