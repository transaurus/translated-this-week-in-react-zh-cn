## ⚛️ React {#react}

- 💸 [Convex 发布 MCP 服务器](https://convex.link/thisweekmcp)
- 🔐 [React Router 漏洞及 v7.5.2 修复](https://github.com/remix-run/react-router/blob/main/CHANGELOG.md#v752): React Router v7 存在两个可能导致拒绝服务攻击和跨站脚本攻击的重要安全漏洞，最新 v7.5.2 版本已修复。多家云服务商（Vercel/Netlify/Cloudflare）已部署缓解措施，但仍建议升级。
- 👀 [Next.js 文档 PR - 移除 Vercel 相关提及](https://github.com/vercel/next.js/pull/78557): Next.js 文档将减少对 Vercel 的倾向性，更公平地展示其竞品。笔者认为开源项目通过付费服务实现可持续无可厚非，但需重建部分用户的信任。需明确 Vercel 并未"接管"React 团队，上周 React Labs 宣布的客户端功能（如`<ViewTransition>`）由[Vercel 员工实现](https://bsky.app/profile/ricky.fm/post/3lnvyuennss2b)。
- 👀 [Vite RSC 计划](https://github.com/hi-ogawa/vite-plugins/issues/748): 某 Vite 贡献者正[接手开发](https://github.com/facebook/react/pull/31768#issuecomment-2823466684)未来的`react-server-dom-vite`包，已向 Waku/RedwoodSDK 等框架提交集成 PR。
