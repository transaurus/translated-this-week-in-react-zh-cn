## ⚛️ React核心 {#react}

[![React服务端操作能彻底解决表单问题吗？](/emails/issues/156/serveractions.jpg)](https://www.mux.com/blog/what-are-react-server-actions)

[**React服务端操作能彻底解决表单问题吗？**](https://www.mux.com/blog/what-are-react-server-actions)

本文深入浅出地介绍了即将推出的React功能——服务端操作（已作为实验性功能集成至Next.js 13.4）。包含视频标题上传的简短教程。虽然服务端操作能返回响应为用户提供反馈，但目前开发者体验仍有优化空间。

在我看来，Server Actions 应当致力于全面支持渐进式增强。这可能需要引入一个"flash" API 来管理用户反馈，而无需任何客户端 JavaScript。许多后端框架已实现此类功能，Remix 就提供了 ([`session.flash()`](https://remix.run/docs/en/main/utils/sessions#sessionflashkey-value)) API。您可能感兴趣的相关信息：通过 [`action.$$FORM_ACTION`](https://twitter.com/sebastienlorber/status/1689921046856474625) 可以扩展 React 的渐进式增强行为。
