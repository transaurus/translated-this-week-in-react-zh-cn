## 📱 React-Native {#react-native}

[![React Native WebGPU](/emails/issues/196/rn-webgpu.jpg)](https://www.youtube.com/watch?v=i-DGY98cTjk)

**🎥 [React Native WebGPU](https://www.youtube.com/watch?v=i-DGY98cTjk)**

React Native WebGPU正式发布！这项技术通过复用Web API标准，为React Native提供了调用GPU进行高级图形渲染与计算的统一接口。开发者可直接集成现有WebGPU代码示例和类库（如ThreeJS）到React Native应用中。
除了图形处理，React Native WebGPU还能用于矩阵运算等场景，为设备端机器学习算法提供了新可能。该技术采用Nitro Modules实现TypeScript到C++的无缝互操作，目前处于技术预览阶段，预示着激动人心的技术前景！

---

**React Native模块的未来演进**

React Native开发者迎来重大革新！Marc Rousavy推出的NitroModules彻底改变了原生代码编写方式📦 [性能测试全面超越现有方案](https://github.com/mrousavy/NitroBenchmarks)。他在VisionCamera项目中发现了JSI的局限性。目前C++端已趋稳定，正被应用于🐦 [react-native-unistyles](https://x.com/jpudysz/status/1822541758137303233)和react-native-webgpu等项目，Swift与Kotlin绑定仍在开发中。

👀 [与此同时，一份推动Swift开发TurboModules的新提案已提交](https://github.com/okwasniewski/discussions-and-proposals/blob/22edd2c7853aaf2ee654b57f4521e43d668103fb/proposals/0000-swift-turbo-modules.md)。通过弃用Objective-C和C++，这些改进将大幅降低原生模块开发门槛。技术升级即将带来更高效的开发体验！
